# Pretreatment nonresponse axis in ulcerative colitis anti-TNF cohorts

This repository contains the public reproducibility package for a transcriptomic analysis of pretreatment mucosal states associated with anti-TNF nonresponse in ulcerative colitis.

The package includes analysis scripts, derived result tables, exported figures, supplementary tables, and provenance files. It does not include manuscript drafts, submission files, local workflow notes, credentials, or raw expression matrices.

## Repository layout

- `scripts/`: data-processing, statistical analysis, and figure-generation scripts
- `results/`: derived analysis tables used by the figures and supplementary tables
- `plots/publication/`: exported main and supplementary figures in PNG and PDF
- `supplementary_tables/`: supplementary XLSX tables S0-S7
- `docs/`: data manifest, figure/table provenance, statistical rules, and rendered supplementary tables

## Quickstart

The quickstart mode uses the included derived tables and regenerates the exported figures and supplementary-table rendering.

```bash
python3 -m pip install -r requirements.txt
Rscript scripts/install_r_deps.R --figures-only
bash scripts/reproduce_one_click.sh
```

Expected outputs:

- `plots/publication/Figure1_study_design.{png,pdf}`
- `plots/publication/Figure2_discovery_pathways.{png,pdf}`
- `plots/publication/Figure3_replication_forest.{png,pdf}`
- `plots/publication/Figure4_singlecell_localization.{png,pdf}`
- `plots/publication/Figure5_specificity.{png,pdf}`
- `plots/publication/FigureS1.{png,pdf}`
- `plots/publication/FigureS3_singlecell_signature_heatmaps.{png,pdf}`
- `supplementary_tables/TableS7_gene_set_overlap_and_singlecell_tests.xlsx`
- `docs/SUPPLEMENTARY_TABLES.md`

`FigureS2` requires rebuilt sample-level core matrices because it uses within-cohort inflammatory and EMT proxy scores. It is regenerated by the full public-data mode below and is also included in `plots/publication/`.

## Full re-run from public GEO matrices

This mode downloads public GEO series matrices, rebuilds the core bulk analysis, recomputes the GSE92415 sensitivity analysis, regenerates figures, and refreshes the supplementary-table rendering.

```bash
Rscript scripts/install_r_deps.R --full
bash scripts/reproduce_one_click.sh --from-public-data
```

The full mode writes downloaded GEO matrices under `data/raw/geo/matrix/` and intermediate processed files under `data/processed/`. These directories are intentionally ignored by git.
