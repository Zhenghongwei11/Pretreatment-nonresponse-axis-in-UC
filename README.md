# UC Infliximab Primary Nonresponse (Public Reproducibility Package)

This repository is a reviewer-facing reproducibility package for a public-data analysis of pretreatment mucosal transcriptomes in ulcerative colitis (UC) patients treated with infliximab, focusing on primary nonresponse.

The package contains:
- `scripts/`: analysis and figure-generation scripts
- `results/`: derived tables used to support figures (no raw patient-level expression matrices)
- `plots/publication/`: exported publication figures (PNG + PDF)
- `docs/`: data manifest, figure provenance, and predeclared statistical rules

## Quickstart (regenerate figures from included tables)

This mode does not download any external data. It regenerates Figures 2–5 from the included `results/` tables.

```bash
Rscript scripts/make_publication_figures.R
```

Outputs:
- `plots/publication/Figure2_discovery_pathways.{png,pdf}`
- `plots/publication/Figure3_replication_forest.{png,pdf}`
- `plots/publication/Figure4_singlecell_localization.{png,pdf}`
- `plots/publication/Figure5_specificity.{png,pdf}`

Note: `plots/publication/Figure1_study_design.{png,pdf}` is provided as a fixed schematic and is not regenerated by the pipeline.

## Optional: full re-run from public bulk GEO data

This mode downloads the GEO series matrices for the three bulk cohorts and rebuilds the derived tables under `results/`.

```bash
bash scripts/reproduce_one_click.sh --from-public-data
```

## Environment

Recommended:
- R >= 4.4
- Python >= 3.10

See:
- `requirements.txt` (Python)
- `scripts/install_r_deps.R` (R)

Install dependencies:
```bash
python3 -m pip install -r requirements.txt
Rscript scripts/install_r_deps.R --figures-only
```

Optional (for a full bulk re-run from public GEO matrices):
```bash
Rscript scripts/install_r_deps.R --full
```
